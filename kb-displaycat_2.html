搬瓦工如何搭建ssr高速节点？

1、购买服务器：首先，您需要购买一台具有良好带宽和运行速度的服务器。推荐使用云服务器，例如AWS EC2或DigitalOcean等。

2、安装系统：根据您的服务器选择安装相应的操作系统，如CentOS或Ubuntu等。

3、安装必要软件：运行以下命令以安装必要的软件：

 
        yum update -y
        yum install git -y
        yum groupinstall "Development Tools" -y
        yum install epel-release -y 
        yum install python-setuptools m2crypto cymysql git -y

4、下载SSR：从github上下载SSR源代码。

 git clone -b manyuser https://github.com/shadowsocksr-backup/shadowsocksr.git

5、配置SSR：进入shadowsocksr目录，并按照你的需求编辑config.json文件。修改加密方式、端口、密码以及协议等参数。

6、启动SSR：运行以下命令启动SSR。

 
            cd shadowsocksr/
            python server.py

7、优化SSR性能：可进行一些优化设置，如开启BBR拥塞控制算法等，以提高SSR性能和速度。

完成以上步骤后，您就可以在本地设备上配置SSR客户端，并连接到您的SSR服务器了。
